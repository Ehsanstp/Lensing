## Gravitational Lense Detection Using CNN

This project explores the use of Convolutional Neural Networks (CNNs) for detecting gravitational lenses in astronomical data. It was developed as part of a learning initiative to better understand the underlying structures and behavior of neural networks.

The model and experiments were implemented using Google Colab, with AI-assisted debugging and iteration to improve development efficiency.

In addition to deep learning methods, the project uses simple, conventional methods for better identification process and distinguishing true gravitational lensing events from false positives.
